# Taylor Swift Club 前端

这是 Taylor Swift Club 网站的前端代码仓库。

## 项目结构

```
frontend/
├── index.html      # 首页
├── forum.html      # 论坛页面
├── store.html      # 商店页面
├── admin.html      # 管理页面
└── disclaimer.html # 免责声明
```

## 部署说明

本项目使用 Cloudflare Pages 进行部署。

### 部署步骤

1. 在 Cloudflare Pages 中创建新项目
2. 连接 GitHub 仓库
3. 配置构建设置：
   - 构建命令：留空
   - 输出目录：/
